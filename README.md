# wynnjs

TypeScript tooling for the [Wynncraft API](https://docs.wynncraft.com/).

## Packages

| Package                         | Description                       |
| ------------------------------- | --------------------------------- |
| [`@wynnjs/api`](./packages/api) | Typed client for Wynncraft API v3 |

## Apps

| App                           | Description             |
| ----------------------------- | ----------------------- |
| [`@wynnjs/docs`](./apps/docs) | VitePress documentation |

Docs: [jakehwll.github.io/wynnjs](https://jakehwll.github.io/wynnjs/)

## Development

```bash
bun install
bun run ci
bun run docs
```

See [CONTRIBUTING.md](./CONTRIBUTING.md) for setup, conventions, and release notes.
